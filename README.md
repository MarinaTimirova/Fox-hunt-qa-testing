# Fox-hunt-qa-testing
Testing "Fox Hunt" mobile game
**О проекте**
Тестирование мобильной игры-головоломки "Охота на лис". Эта игра базируется на механике игры "Сапёр". Суть игры: найти лис на игровом поле.
**Содержание докуметна**
1. Test plan (Тест план)
2. Test suit (Набор проверок)
3. Bug repots with video and screenshorts (Отчеты об ошибках с приложеными фото и видео)
